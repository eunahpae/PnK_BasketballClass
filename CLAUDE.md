## ⭐ 핵심 기능 명세

### 1. 수강 신청 시스템

**신청 폼 항목**
: 이름 / 연락처 / 학년 / 희망 반 선택 / 기존 수강 여부 / 개인정보 동의

**신청 흐름 (사용자)**

```
수강 신청 페이지 진입
  → 희망 반 선택 (잔여석 실시간 표시)
  → 잔여석 있음: 신청 폼 작성 → 신청 완료 → 카카오 알림톡 발송
  → 잔여석 없음: 대기자 등록 폼으로 자동 전환 → 순번 안내 → 카카오 알림톡 발송
```

**대기자 처리 흐름**

```
대기자 등록
  → 순번 부여 + 알림톡 발송 (대기 순번, 예상 연락 일정)
  → 자리 발생 시 1순위 대기자에게 자동 알림톡 발송 (신청 링크 포함)
```

> ⚠️ Phase 1에서는 잔여석 실시간 연동 없이 오프라인 확인 방식으로 오픈.
> ⚠️ 잔여석 실시간 연동 + 대기자 자동 알림은 Phase 2에서 구현.

**개인레슨 신청 폼 별도 제공**
: 코치 선택 / 희망 날짜 및 시간 입력

**관리자 알림**
: 신청 완료 시 관리자에게 문자 + 이메일 동시 발송

### 2. 수업 안내 페이지

**정규반 시간표**

| 표시 항목 | 내용 |
|-----------|------|
| 표 구성 | 요일 / 시간 / 대상 학년 / 정원 / 잔여석 / 수강료 |
| 마감 처리 | 잔여석 0 → '마감' 표시 + 대기 신청 버튼으로 전환 |
| 상세 팝업 | 반별 커리큘럼, 준비물 등 상세 설명 팝업 제공 |

**개인레슨**
- 코치별 프로필 카드 (사진, 경력, 전문 포지션)
- 가능 시간 캘린더 표시
- 개인레슨 신청 폼 연결

**특강 / 캠프 / 대회**
- 캘린더 + 카드 리스트 혼합 형태
- 이벤트 클릭 시 상세 안내 페이지 (일정, 비용, 신청 방법)
- 신청 마감일 D-Day 카운터 표시

### 3. 관리자 백오피스

비개발자인 원장님이 직접 관리할 수 있도록 직관적인 UI로 구성.
모든 관리 기능은 코드 조작 없이 UI에서 처리 가능해야 함.

| 기능 | 상세 내용 |
|------|-----------|
| 수강생 관리 | 신청 목록 조회, 상태 변경 (승인/대기/취소), 수강생 정보 수정 |
| 출결 관리 | 날짜별 출석 체크, 결석 사유 메모, 학부모 알림 발송 |
| 시간표 관리 | 반 추가/수정/삭제, 정원 설정, 잔여석 자동 계산 |
| 공지사항 관리 | 게시글 작성/수정/삭제, 상단 고정 설정, 카테고리 관리 |
| 갤러리 관리 | 사진/영상 업로드, 캡션 편집, 순서 변경, 삭제 |
| 문의 관리 | 문의 목록 조회, 답변 작성, 알림톡 발송 확인 |
| 알림 발송 | 카카오 알림톡 수동 발송 (휴강, 변경 사항 안내) |
| 통계 대시보드 | 월별 신청자 수, 반별 수강 현황, 문의 건수 차트 |

### 4. 카카오 알림톡 자동 발송

| 발송 시점 | 수신자 | 내용 |
|-----------|--------|------|
| 수강 신청 완료 | 신청자 | 신청 완료 확인, 반 정보, 담당 코치 |
| 대기자 등록 | 대기자 | 대기 순번, 예상 연락 일정 |
| 자리 발생 | 대기자 1순위 | 수강 신청 가능 안내 + 링크 |
| 수업 전날 | 수강생·학부모 | 내일 수업 시간 리마인드 |
| 결석 처리 | 학부모 | 자녀 결석 확인 안내 |
| 문의 답변 완료 | 문의자 | 답변 등록 알림 |
| 휴강/일정 변경 | 해당 반 전체 | 변경 내용 및 보강 일정 |

> ⚠️ 알림톡 자동 발송은 Phase 2에서 구현. Phase 1에서는 관리자 수동 발송만 지원.
> ⚠️ 연동 API: 채널톡 또는 NHN Cloud 알림톡 API (개발사 확정 후 공지).
> ⚠️ 알림톡 발송 비용: 건당 약 10~15원 (발송 건수에 따라 변동).

### 8. 농구 정보 게시판

원장님이 직접 작성하는 블로그형 콘텐츠 게시판.
네이버·구글 검색 유입을 통한 신규 수강생 자연 유치 목적.

**카테고리 구성**

| 카테고리 | 콘텐츠 예시 | 주요 대상 |
|----------|-------------|-----------|
| 드릴·훈련 방법 | 드리블 기초 훈련법, 슛폼 교정, 혼자 할 수 있는 연습 | 수강생·학부모 |
| 농구 규칙·용어 | 바이얼레이션 종류, 포지션별 역할, 심판 시그널 | 입문자·학부모 |
| 포지션별 전술 가이드 | 가드·포워드·센터 전술, 수비·공격 개념, 팀플레이 | 중급 이상 수강생 |
| NBA·KBL 뉴스 | 주요 선수 소식, 경기 하이라이트, 시즌 일정 | 전 연령 |
| 대회 정보·일정 | 지역 유소년 대회, 학교 대회, 참가 신청 방법 | 수강생·학부모 |
| 장비·용품 추천 | 농구화 고르는 법, 포지션별 용품 추천, 관리법 | 수강생·학부모 |

**목록 화면**
- 카테고리 탭 필터 (전체 / 드릴·훈련 / 규칙·용어 / 전술 / NBA·KBL / 대회 / 장비)
- 카드형 레이아웃: 썸네일 + 제목 + 카테고리 뱃지 + 작성일 + 조회수
- 인기글 상단 노출 또는 '추천' 뱃지 표시
- 제목·본문 키워드 검색 기능

**글 상세 화면**
- 제목, 카테고리, 작성일, 조회수 표시
- 본문: 텍스트 + 이미지 + 유튜브 영상 임베드 지원
- 이전 글 / 다음 글 이동 버튼
- 카카오·인스타그램·링크 복사 공유 버튼
- 같은 카테고리 관련 글 하단 추천 (3건)

**관리자 작성 기능**
- 비주얼 에디터(WYSIWYG) 제공 — HTML 없이 블로그처럼 작성
- 썸네일 이미지 업로드, 카테고리·태그 선택
- 임시저장 및 예약 발행 기능 (날짜 지정 자동 공개)
- 글 발행 시 카카오채널 공유 연동 (선택 옵션)

**SEO 설정**
- 글마다 메타 제목·설명 입력 가능 (Yoast SEO 플러그인 적용)
- URL 구조: `/basketball-info/카테고리/글제목` 형태 설정
- 이미지 alt 태그 입력 안내로 검색 노출 강화
- 네이버 블로그·인스타와 중복 방지 — 홈페이지 우선 발행 권장

> ⚠️ Phase 1: 기본 게시판 구현 / Phase 2: SEO 고도화

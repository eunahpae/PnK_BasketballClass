## 🎨 디자인 가이드

- 컬러: 미확정 (디자인 시안 확정 후 업데이트)
- 폰트: 미확정
- 반응형 기준: Mobile 360px / PC 1920px
- 팝업: 모바일에서 화면 전체를 가리지 않도록 처리
- 아이콘 사용 최소화: 텍스트 레이블로 대체 우선, 
  꼭 필요한 경우에만 사용 (예: 검색, 닫기, 메뉴햄버거)
- 아이콘 라이브러리 import 지양 (lucide-react, font-awesome 등)
- 불가피한 경우 SVG 인라인으로 직접 삽입
